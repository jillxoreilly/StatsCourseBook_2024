Sampling with and without replacement#

Toy example#

Let‚Äôs explore the idea of sampling with and without replacement using a very simple example (a simple example designed just to illustrate a point is sometimes called a toy example)

Say I have a dataset listing four children‚Äôs pets:

[cat, dog, cat, rabbit]

If I sample from this dataset, I get a new list of pets. Say I draw a sample of size \(ùëõ=2\), my sample might be [cat, cat] (if I‚Äôm lucky- I love cats!)

Without replacement#

If I sample without replacement, after I have ‚Äòdrawn‚Äô my first sample pet fromt the original dataset, I cannot draw it again - my next sample pet will be drawn from the remaining three. The consequence of this is that all samples of size ùëõ=4 contain all of the original 4 pets, albeit in a different order

[cat, cat, dog, rabbit]

[rabbit, cat, dog, cat]

[cat, dog, rabbit, cat]

etc

With replacement#

If I sample with replacement, each ‚Äòdraw‚Äô can be any of the four animals (think of it like pulling a card from a deck, checking which animal is on it, and then replacing the card in the deck before the next sample is drawn).

So I could get:

[cat, cat, cat, cat]

if I‚Äôm really lucky!

or more likely:

[cat, dog, cat, cat]

[rabbit, dog, cat, rabbit]

‚Ä¶ etc

Let‚Äôs try it, replacing the animals with numbers: dog = 1, cat = 2, rabbit = 3

# Sampling without replacement
data = [2,1,3,2]
nReps= 10

for i in range(nReps):
    sample = np.random.choice(data,4,replace=False)
    print('sample ' + str(i) + ' = ' + str(sample))

sample 0 = [2 1 2 3]
sample 1 = [3 1 2 2]
sample 2 = [1 2 3 2]
sample 3 = [2 2 1 3]
sample 4 = [2 3 1 2]
sample 5 = [1 2 3 2]
sample 6 = [1 2 2 3]
sample 7 = [2 3 2 1]
sample 8 = [2 2 3 1]
sample 9 = [1 3 2 2]

When sampling without replacement each sample should be identical in contents (albeit in a random order) - this is clearer if I sort the sample values in ascending order:

# Sampling without replacement
data = [2,1,3,2]

Real world examples#

A real world example of sampling without replacement would be if we give 100 students a wellbeing questionairre - each student in our sample is a member of the parent distribution (all students/ allstudents in the college we were sampling from etc). We will only want to survey each student once.

A real world of example of sampling with replacement is rolling a dice. Each diceroll yeilds an outcome (1-6) that is a sample from an infinite selection of possible outcomes (think of it as drawing a card from a deck of cards with numbers 1-6 on them, but because there are an infinite number of cards, ‚Äòusing up‚Äô a six doesn‚Äôt reduce the change of the next diceroll being a six).

Sampling from a Pandas dataframe#

In the example above our ‚Äòpets‚Äô were a numpy array, but more often our data are supplied as a dataframe (a table containing columns with text and numbers, with headings etc)

Pandas has a handy built-in sampling function which does a similar job to numpy.random.choice() but for sampling within a Pandas dataframe

Let‚Äôs see it at work:
